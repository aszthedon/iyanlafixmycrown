# Better Admin + Add-on Assignment Version

This version adds:
- add/edit/delete/reorder services
- add-on manager
- per-service add-on assignment
- booking widget only shows the correct add-ons for the selected service
- server-side booking validation only charges allowed add-ons

## Run
```bash
cp .env.example .env
npm install
npm run dev
```

Admin page:
- `http://localhost:3000/admin`
